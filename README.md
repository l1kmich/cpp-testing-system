# C++ Testing System 🚀

Консольная система тестирования на C++, разработанная в рамках курсового проекта. Приложение предоставляет модульную архитектуру для комплексной проверки, валидации и оценки знаний пользователей по языку программирования C++ с разделением вопросов по уровням сложности.

## 📋 Основные возможности

- **Авторизация и сессии:** Разграничение прав пользователей и студентов с безопасным хранением учетных записей (`users.txt`).
- **Многоуровневое тестирование:** Динамическая загрузка вопросов из 10 независимых уровней сложности (`level1.txt` — `level10.txt`).
- **Надежная валидация ввода:** Интегрированный компонент `Validator` полностью исключает падение программы (Runtime Error) при некорректном или слишком длинном вводе пользователя.
- **Оптимизированная логика подсчета:** Архитектура учитывает защиту от переполнения данных при обработке больших числовых значений и накоплении статистики.
- **Кастомизированный UI:** Разделение логики интерфейса через `ConsoleUtils` для создания чистого меню в консоли.

## 🛠 Архитектура проекта

Проект спроектирован по принципам разделения ответственности (Separation of Concerns):

- `CourseProject.cpp` — Точка входа в приложение (`main`).
- `App.h / App.cpp` — Главный оркестратор сессии, управляющий жизненным циклом программы.
- `Models.h / Models.cpp` — Описание сущностей предметной области (Пользователь, Вопрос, Тест, Результат).
- `DataManager.h / DataManager.cpp` — Низкоуровневый модуль для чтения/записи плоских текстовых баз данных.
- `Validator.h / Validator.cpp` — Компонент строгой валидации пользовательских команд.
- `ConsoleUtils.h / ConsoleUtils.cpp` — Утилиты для форматирования текстового вывода и управления экраном.

## 📦 Структура базы данных (Файлы данных)

Программа использует систему плоских файлов для персистентного хранения информации:
- `users.txt` — Профили пользователей и доступы.
- `level1.txt` ... `level10.txt` — Базы структурированных вопросов по категориям сложности.
- `results.txt` — Логи ответов и итоговые баллы участников тестирования.
