# C++ Testing System 🚀

Консольное приложение на C++, разработанное в рамках курсового проекта. Система предназначена для комплексной проверки и оценки знаний пользователей по языку программирования C++ с разделением на уровни сложности.

## 📋 Основные возможности

- **Авторизация и аутентификация:** Разграничение прав пользователей/студентов (хранение данных в `users.txt`).
- **Многоуровневое тестирование:** Поддержка 10 различных уровней сложности (`level1.txt` — `level10.txt`).
- **Валидация ввода:** Безопасный и надежный сбор ответов и команд пользователя без падений программы (`Validator`).
- **Управление данными:** Динамическая загрузка вопросов, парсинг текстовых баз данных и сохранение истории прохождения тестов.
- **Интуитивный консольный интерфейс:** Удобное меню и кастомизированный вывод текстовой информации (`ConsoleUtils`).

## 🛠 Архитектура проекта

Проект построен на принципах модульности и чистого кода (Clean Code):

- `CourseProject.cpp` — Точка входа в приложение (`main`).
- `App.h / App.cpp` — Главный оркестратор логики приложения и жизненного цикла сессии.
- `Models.h / Models.cpp` — Описание сущностей (Пользователь, Вопрос, Тест, Результат).
- `DataManager.h / DataManager.cpp` — Модуль для чтения/записи файлов конфигурации и результатов.
- `Validator.h / Validator.cpp` — Компонент для строгой проверки пользовательского ввода.
- `ConsoleUtils.h / ConsoleUtils.cpp` — Утилиты для форматирования консольного вывода и работы с интерфейсом.

## 📦 Структура базы данных (Файлы данных)

- `users.txt` — База данных зарегистрированных пользователей.
- `questions.txt` / `level1-10.txt` — Банк вопросов, распределенный по категориям сложности.
- `results.txt` — История прохождения тестов с фиксацией итоговых баллов.
